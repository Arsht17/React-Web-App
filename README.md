In order to Run Live Demo in stackBlitz follow this steps:
1. split the console.
2. type "cd Server", click enter.
3. then type "npm install", click enter.
4. after instalation type "npm run", click enter.
5. then type "node index.js", click enter.
6. Finally you can reload the previw or control + left mouse click on the port link in the left cosole.
Live Demo will load the app.
